# FurinaXGP
Multi functional tool that has some things built in useful for XGP and altshops(?) | Now with a config file!!!!

# Config

furina_config.json.
Example:
```
{
    "driver_path": "String",
    "XAGtoken": "String",
    "xbltoken": "String",
    "minTypeSpeed": 0.2, // Float
    "maxTypeSpeed": 0.6, // Float
    "defaultTimeoutinSeconds": 1000 // INT
}
```
